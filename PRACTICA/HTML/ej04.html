Ejercicio 04

Realice una página web con ocho párrafos de 50 palabras cada uno y un ancho del 50% y un alto de 30%, con un borde para cada estilo.

Este es el primer párrafo, con borde sólido. Aquí se puede ver cómo se muestra el estilo solid en CSS. Cada párrafo de este ejercicio tiene exactamente cincuenta palabras para cumplir con el requerimiento. El ancho es del cincuenta por ciento y el alto es del treinta por ciento de la ventana.

Este es el segundo párrafo, con borde dashed. El estilo dashed crea una línea de borde discontinua alrededor del párrafo.

Este es el tercer párrafo, con borde dotted. El estilo dotted crea un borde punteado alrededor del párrafo.

Este es el cuarto párrafo, con borde double. El estilo double crea un borde doble alrededor del párrafo.

Este es el quinto párrafo, con borde groove. El estilo groove crea un efecto de relieve hundido en el borde del párrafo.

Este es el sexto párrafo, con borde ridge. El estilo ridge crea un efecto de relieve elevado en el borde del párrafo.

Este es el séptimo párrafo, con borde inset. El estilo inset crea un efecto de inserción en el borde del párrafo.

Este es el octavo párrafo, con borde outset. El estilo outset crea un efecto de salida en el borde del párrafo.
