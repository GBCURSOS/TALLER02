Ejercicio 05

Realice una página web que tenga tres párrafos, cada uno de 50 palabras, ancho del 50%, alto automático y el primero esté centrado, el segundo pegado a la derecha y el tercero a la izquierda. Todos deben tener un color de fondo y texto diferente. El fondo de la página debe ser diferente. Todos los párrafos deben presentar borde de 2px de estilo y color diferente.

Este es el primer párrafo, centrado en la página. Tiene un fondo amarillo claro y el texto en azul petróleo. El borde es sólido y de color rojo brillante.

Este es el segundo párrafo, alineado a la derecha. El fondo es azul claro y el texto es de color vino. El borde es dashed y de color azul verdoso. El ancho es del cincuenta por ciento y el alto es automático.

Este es el tercer párrafo, alineado a la izquierda. El fondo es verde claro y el texto es rojo suave. El borde es dotted y de color rojo oscuro. El ancho es del cincuenta por ciento y el alto es automático.
