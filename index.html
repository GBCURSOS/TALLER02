Estilo Descripción
solid Borde sólido, una línea continua.
dashed Borde discontinuo, formado por guiones.
dotted Borde punteado, formado por puntos.
double Borde doble, dos líneas sólidas paralelas.

Manejo de Colores

Cuando se tiene una paleta de cuatro colores, el criterio típico para usarlos en una página web es:

  • Color 1: Se utiliza como color principal o de acento (por ejemplo, encabezados, botones, enlaces destacados o bordes importantes).
  • Color 2: Suele emplearse como color de fondo principal o secundario (por ejemplo, fondo de la página o de secciones).
  • Color 3: Se usa para el texto principal o secundario, o como color de contraste para elementos destacados.
  • Color 4: Se reserva para detalles, bordes, hover, elementos interactivos o fondos alternativos.
La asignación exacta puede variar según la armonía visual, el contraste y la legibilidad, pero siempre se busca que los colores tengan suficiente contraste entre fondo y texto, y que la combinación sea agradable y coherente visualmente.
